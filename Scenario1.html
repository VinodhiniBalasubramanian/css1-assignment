This is a training project which includes 3 exercises using HTML and CSS. CSS is used to control the style and layout of a web page and CSS3 is the latest standard/version of CSS. You can create 3 separate html pages to complete all 3 scenarios/excercises as instructe.

Prerequisite: User should have basic knowledge in HTML and CSS3. For development purpose, user may use any of web page editor or IDE like notepad ++, eclipse. NetBeans etc. on local environment.

Once the project work completed user need to submit the project to TopGear repository for a review process. For joining/submiting the project user can refer relavant guidelines provided by TopGear communite or contact at TopGear center.
